# Kinemaths Website (HTML/CSS/JS/Python)

Simple homepage starter for the Kinemaths web game.

## Run locally

```bash
cd /Users/desha/Documents/projects/Kinemaths
python3 serve.py
```

Then open:

`http://127.0.0.1:8000`

## Hosting later

This is a static site (`index.html`, `styles.css`, `script.js`), so you can host it on any static host:
- GitHub Pages
- Netlify
- Vercel (static)
- Cloudflare Pages
